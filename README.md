# MM-MathAlign

基于 SFT 与强化学习的 Qwen 多模态数学推理对齐项目。

## 项目简介

MM-MathAlign 是一个面向 **多模态数学推理（Multimodal Mathematical Reasoning）** 的研究项目，基于 Qwen 大模型进行能力对齐与增强。

本项目利用多模态数学数据（图像 + 文本 + 公式），通过以下两阶段训练流程提升模型的数学推理能力：

- **监督微调（Supervised Fine-Tuning, SFT）**
- **强化学习对齐（Reinforcement Learning, RL）**

目标是提升大模型在复杂数学问题上的理解、推理和生成能力。

## 项目特点

- 支持 **多模态数学数据训练**
- 完整的 **SFT 训练流程**
- 基于 **强化学习的模型对齐**
- 支持 **数学推理任务评测**
